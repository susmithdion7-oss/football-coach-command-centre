# Coach Command Centre

Coach Command Centre is a football coaching management tool for head coaches.

Version 0.1 focuses on four core features:

- Player profiles
- Training session planning
- Basic tactical board
- PDF export

This project is currently in the MVP stage.

The first version will not include:

- Player login
- Team chat
- Cloud database
- AI generation
- Payment system
- Mobile app

The goal of version 0.1 is to prove that a coach can manage players, plan a session, create a tactical diagram, and export a useful PDF from one place.

For the full product plan, see `PRODUCT_BRIEF.md`.
